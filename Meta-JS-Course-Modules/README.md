# Meta-JavaScript-Course-Practice

Mini Projects that reinforce Programming with JavaScript concepts

## Concepts Covered

- Variables and primitive data types (string, number, boolean)
- Arithmetic operators (+, -, \*, /)
- String-to-number conversion
- String concatenation and length
- Boolean logic and toggling states

### Module 1: Variables and Primitive Data Types

This module contains practice exercises for:

- Displaying and updating variable values
- Performing arithmetic calculations
- Combining and manipulating strings
- Toggling boolean states (lights, doors)
