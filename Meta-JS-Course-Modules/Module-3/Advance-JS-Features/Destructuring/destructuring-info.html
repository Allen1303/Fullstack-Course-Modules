What is Destructuring?

Destructuring is a JavaScript expression that allows you to extract multiple values from arrays or objects and assign them to variables in a single statement. It's like unpacking a box and taking out exactly what you need, leaving the rest behind.
üì¶ Basic Array Destructuring 
// Traditional way
const colors = ['red', 'green', 'blue'];
const first = colors[0];
const second = colors[1];

// With destructuring - much cleaner!
const [first, second, third] = ['red', 'green', 'blue'];
console.log(first, second, third); // Output: red green blue
üèóÔ∏è Basic Object Destructuring 
// Traditional way
const person = { name: 'Alice', age: 30, city: 'New York' };
const name = person.name;
const age = person.age;

// With destructuring
const { name, age, city } = { name: 'Alice', age: 30, city: 'New York' };
console.log(`${name} is ${age} years old and lives in ${city}`);
// Output: Alice is 30 years old and lives in New York

‚ö° Performance & Readability Example 
// Without destructuring - repetitive and harder to read
function displayUser(user) {
  console.log('Name: ' + user.name);
  console.log('Email: ' + user.email);
  console.log('Role: ' + (user.role || 'guest'));
}

// With destructuring - clean and efficient
function displayUserClean({ name, email, role = 'guest' }) {
  console.log(`Name: ${name}`);
  console.log(`Email: ${email}`);
  console.log(`Role: ${role}`);
}

const user = { name: 'John Doe', email: 'john@example.com' };
displayUserClean(user); // Output: Name: John Doe, Email: john@example.com, Role: guest

How Destructuring Works

Destructuring uses pattern matching to extract values. The left side of the assignment defines the pattern, and JavaScript matches it against the right side structure. Let's explore different patterns and advanced techniques:
üéØ Advanced Array Destructuring 
// Skip elements, use rest operator, set defaults
const numbers = [1, 2, 3, 4, 5];
const [first, , third, ...rest] = numbers;
console.log(first); // Output: 1
console.log(third); // Output: 3
console.log(rest); // Output: [4, 5]

// Default values for missing elements
const [a = 10, b = 20, c = 30] = [1];
console.log(a, b, c); // Output: 1 20 30

// Swapping variables elegantly
let x = 100, y = 200;
[x, y] = [y, x];
console.log(x, y); // Output: 200 100
üèóÔ∏è Advanced Object Destructuring 
// Renaming variables, nested destructuring, defaults
const data = {
  id: 123,
  profile: {
    firstName: 'Sarah',
    lastName: 'Johnson',
    social: {
      twitter: '@sarahj'
    }
  }
};

// Rename variables and set defaults
const { 
  id: userId, 
  profile: { 
    firstName: name, 
    social: { twitter, instagram = 'Not provided' } 
  } 
} = data;

console.log(userId); // Output: 123
console.log(name); // Output: Sarah
console.log(twitter); // Output: @sarahj
console.log(instagram); // Output: Not provided

When to Use Destructuring

Knowing when to apply destructuring is crucial for writing effective JavaScript. Here are common scenarios where destructuring shines and makes your code more professional:

üåê API Response Handling

Extract specific fields from complex API responses without accessing nested properties repeatedly.

üì• Function Parameters

Create flexible functions that accept objects as parameters, making them more readable and maintainable.

üì§ Function Returns

üîÑ Variable Swapping

‚öôÔ∏è Configuration Objects

üìä Data Processing

Work with arrays of objects, extracting specific properties for calculations or transformations.

üåê Real-World Examples 
// 1. API Response Handling
const apiResponse = {
  status: 200,
  data: {
    users: [
      { id: 1, name: 'Alice', email: 'alice@example.com' },
      { id: 2, name: 'Bob', email: 'bob@example.com' }
    ]
  }
};

const { data: { users }, status } = apiResponse;
console.log(`Status: ${status}, Found ${users.length} users`);

// 2. Function with destructured parameters
function createUser({ name, email, role = 'user', active = true }) {
  return { name, email, role, active, id: Date.now() };
}

const newUser = createUser({ name: 'Charlie', email: 'charlie@example.com' });
console.log(newUser);

// 3. Processing array of objects
const products = [
  { name: 'Laptop', price: 999, category: 'Electronics' },
  { name: 'Book', price: 29, category: 'Education' }
];

products.forEach(({ name, price }) => {
  console.log(`${name}: $${price}`);
});

// 4. Multiple return values
function getCoordinates() {
  return [10, 20];
}

const [x, y] = getCoordinates();
console.log(`Position: (${x}, ${y})`); // Output: Position: (10, 20)
